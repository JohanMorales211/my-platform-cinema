Tecnologías utilizadas

Este proyecto ha sido construido utilizando Angular, aprovechando sus capacidades para crear una aplicación de página única (SPA) moderna y reactiva. Algunas de las características clave incluyen:

  • Componentes reutilizables y modulares.
  • Enrutamiento dinámico para una navegación fluida.
  • Consumo de API externas (TMDB) para obtener datos de películas.
  • Diseño responsive adaptado a diferentes dispositivos.
  • Prácticas modernas de desarrollo front-end.
